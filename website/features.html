Upcoming Features

In order of priority

Documentation

This project is currently severely underdocumented. I plan on writing a good amount of documentation for all sub-projects

Custom Event Handlers

A custom event handler will allow you to add your own functions to run on specific player events (mouse clicks, keypress, etc.)

Better Meshes

Triangle mesh objects currently require a lot of work to be usable.

Image type support

Custom engines for more image filetypes like JPG and BMP to allow for more flexible texturing

Audio

Engines for audio filetypes like MP3 and WAV

Improved Zest

Zest currently functions as almost a configuration language. I'd like to upgrade it to function as a full-fledged programming language.

3D Models

Support for rendering custom OBJ files (or other 3D object file formats)

ASM-WS Improvements

Functionality like reading POST request data, handling PUT, DELETE, etc. methods, and executing files on GET methods
